# 爬蟲寵愛+ 每頁 Block List（以首頁手繪圖為主）

---

## 1. 全域規則
### 必寫規則
- 首頁為可下滑長頁
- 底部導覽列固定可見
- 內容在底部導覽列上方滾動
- 所有子頁面返回按鈕固定在左上角
- 所有子頁面支援左滑返回
- 所有上一頁邏輯都應回到剛剛的頁面，而不是固定某個模組首頁
- Mock 最外層需加 iPhone 框

---

## 2. 首頁主畫面
### 必畫 Blocks
- Header
- Today Widget
  - 今日任務卡片
  - 首屏只露 2 筆
  - 超過 2 筆時需在該區塊內直向下滑
  - 卡片內容：頭像 / 寵物名 / 任務類型 / 時間 / 狀態
  - 點卡片預設進該寵物個體頁
- Today Widget 延伸入口
  - 寵 AI+ 助手（寬橫按鈕)
- 第一排快捷
  - 我的爬寵
  - 餵食／紀錄
- 第二排快捷
  - 餵食計畫
  - 繁孕紀錄
  - 爬寵相簿
- 日曆 Block
  - 整個 4 月月曆
  - 上方圖例：紅點＝餵食、藍點＝其他
  - 餵食事件紅點
  - 其他事件藍點
- 討論 Block
- 商店 Block（排在更下面，需下滑才看到）
- 近期待辦區塊
- 底部導覽列
